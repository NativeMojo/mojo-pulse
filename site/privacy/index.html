Mojo Pulse / Privacy

Privacy

Private by architecture.

Pulse monitors your Mac on your Mac. There's no account, no analytics, no telemetry, and no server of ours holding your data. This page is the whole policy, in plain English.

The default is zero

Everything Pulse observes — metrics, events, security findings, network history, speed-test results — is stored in a local database on your Mac and stays there. There is no sync, no cloud backup of it, and no way for us to see it.

The complete list of network calls

Every network request the app can ever make, and when:

  • Update checks (default on). Pulse asks GitHub whether a newer signed release exists, via the standard Sparkle framework. No personal data rides along.
  • Speed tests (when you run one). Measurement traffic flows to Apple's public measurement servers (Cloudflare as fallback) — that's what a speed test is.
  • Connection-map locations (opt-in, off by default). If you turn on locations, the public IP addresses your Mac talks to are sent to our lookup service to resolve city and risk flags. Local addresses never leave your Mac, and results are cached locally.
  • Domain & IP lookup tools (when you use them). The domain or address you type is sent to our lookup service to fetch its records. That's the point of the tool.
  • App Store seller checks (when you use them). App identifiers are checked against Apple's public catalog to confirm the seller behind an app.

That's the entire list. No analytics SDK, no crash reporter, no advertising identifiers, no fingerprinting, nothing "anonymized."

Permissions, only in context

Pulse requires no permissions to do its core job, and never asks for an administrator password. Some features ask for a system permission the first time you use them, and simply stay off if you decline:

  • Location — macOS requires it just to read Wi‑Fi network names. Your location itself is never used or sent.
  • Local Network — to name devices on your own LAN.
  • Bluetooth — for the sonar, only while its window is open.
  • Notifications — so alerts can reach you.

Your data's lifetime

Delete the app and the data is yours to remove with it — Pulse installs no background helpers or daemons, and its files live in the standard per-app folders macOS cleans up with any uninstaller (or by hand). Nothing persists on any server, because nothing was ever on one.

Verify, don't trust

The complete source code is public on GitHub under the Apache 2.0 license, and every release is signed and notarized by Apple. If this page and the code ever disagree, the code wins — and we'd like to know: open an issue.
