Processes

Every process, explained.

Activity Monitor shows you a thousand rows and wishes you luck. Pulse groups the noise, verifies who's really behind each binary, and gives every process a full dossier one click away.

Real screenshot from the current release. Click to zoom.

Grouped like you actually think

Chrome is one thing to you, so it's one row here — helpers folded underneath with a real combined total, instead of a hundred identical rows hiding the truth. Tabs cut to Apps, Background, System, or Unverified when you're hunting.

Verified publishers, not self-reported names

A column most tools don't have: the legal name from each binary's code signature. Anything can call itself “Google Chrome Helper”; only Google can sign as Google LLC. App Store apps can be double-checked against Apple's public catalog.

A dossier for anything suspicious

The inspector goes as deep as you want: overview, signing and notarization, live connections, open files, loaded modules, environment. And when the verdict is bad, Quit is right there — no round-trip to another app.

Under the hoodRefreshes every 4 seconds, cost-gated so heavy inspection never piles up. Code-signing and notarization checks run on device.
